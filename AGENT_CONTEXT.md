# Agent Mission: Fråga Saga (fragasaga.se) Growth

## 1. Mål (Goal)
- Göra fragasaga.se till NUMMER 1 på Google för offert- och fakturagranskning i Sverige, och etablera Saga som thought leader/auktoritet inom konsument- och företagsskydd vid hantverkar- och leverantörsoffert.
- Driva organisk trafik OCH generera leads (uppladdade offerter/analyser med kontaktuppgifter).

## 2. Produkt
Gratis AI-tjänst: ladda upp offert/faktura → Saga analyserar priset mot svensk marknadsdata och kontrollerar F-skatt och Bolagsverket. Lägen: quote / compare / invoice. Stack: Next.js (App Router) + Supabase.

## 3. Mätbara KPI:er
- Topp 3 → topp 1 på Google för "är offerten rimlig", "granska offert hantverkare", "kolla faktura företag", "ROT-avdrag offert", "F-skatt kontroll" inom 6 månader.
- +20 % organisk trafik per månad.
- 100 kvalificerade analyser/leads per månad.

## 4. Strategi & Taktik
- **Content/SEO:** Bygg djupa guider kring offertgranskning, rimliga priser per hantverkartjänst, ROT/RUT, F-skatt, bluffakturor och vanliga fakturafel. Publicera som App Router-sidor som matchar befintlig struktur (t.ex. app/guider/<slug>/page.tsx).
- **Thought leadership:** Datadrivna inlägg utifrån marknadsdatan Saga redan har.
- **CRO:** Minimera friktion i uppladdnings-/analysflödet; optimera lead-capture.
- **Teknik:** Säkerställ att `npm run build` passerar, snabb LCP, korrekt schema.org (Article/FAQ/WebApplication), komplett sitemap.xml.

## 5. Gränser & Ramverk
- Inga felaktiga prispåståenden; var transparent med datakällor.
- Ge inte juridisk rådgivning — informera.
- Ändra inte kärnflödet eller den visuella profilen utan manuellt godkännande.
- Pusha ALDRIG om `npm run build` misslyckas.
